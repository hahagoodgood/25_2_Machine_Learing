# SeeSpeakHangle 프로젝트 기획서
> **프로젝트 명**: SeeSpeakHangle
> **프로젝트 주제:** 이미지 입력을 받아 자동으로 한국어 설명문(캡션)을 생성하는 AI 모델 개발  
> **데이터셋:** AI Hub 한국어 이미지 설명 데이터셋

---

## 1. 프로젝트 개요

### 1.1 프로젝트 목표
- 입력된 이미지를 분석하여, 해당 이미지를 가장 잘 설명하는 **자연스러운 한국어 문장(캡션)**을 자동으로 생성한다.
- 예시:
  - 입력: 비행기가 활주로에 있는 이미지
  - 출력: “비행기가 공항 활주로 위에 서 있다.”

### 1.2 추진 배경
- 최근 **멀티모달 인공지능(Multimodal AI)** 은 시각 정보와 언어 정보의 통합 학습을 통해 이미지 캡셔닝, 비디오 자막 생성 등 다양한 응용 분야에서 활용되고 있다.
- 본 프로젝트는 이미지-텍스트 융합 모델의 학습 과정을 직접 구현·실험함으로써, **멀티모달 AI**의 핵심 개념과 기술적 구조를 체득하는 것을 목표로 한다.

---

## 2. 데이터셋 정보

| 항목 | 내용 |
|------|------|
| **데이터셋명** | 한국어 이미지 설명 데이터셋 |
| **출처** | AI Hub (https://aihub.or.kr/aihubdata/data/view.do?dataSetSn=261) |
| **구성** | 약 12만 장의 COCO 이미지 경로 + 한국어 캡션 60만 개 |
| **용량** | 약 116MB (텍스트 JSON), 이미지 별도 (COCO 2014 약 19GB) |
| **형식** | JSON: 이미지 파일명 + 한국어 설명문 리스트 |
| **예시** | `"caption_ko": "두 명의 사람이 테니스를 치고 있다."` |
| **참조 데이터** | Microsoft COCO 2014 이미지셋 (train/val) |

## 3. 고려사항
| 구성 | 설명 |
| ----------------- |  -------------------------------------------------- |
| **인코더 (Encoder)** | 이미지 특징 추출 |
| **디코더 (Decoder)** | 텍스트(한국어 캡션) 생성 |
| **토크나이저** | SentencePiece (BPE 기반) or KoNLPy 형태소 단위 토큰화 |
| **손실 함수** | Cross Entropy Loss (Teacher Forcing 적용) |
| **최적화 알고리즘** | Adam Optimizer, Scheduler (ReduceLROnPlateau) |
| **평가지표** | BLEU, METEOR, CIDEr 등 텍스트 생성 성능 지표 |

## 4. 기대효과 및 활용방안
| 항목 | 내용 |
| ---------- | --------------------------------------------------- |
| **기술적 측면** | 이미지 특징 추출, 자연어 생성, 멀티모달 융합 등 핵심 AI 개념 습득            |
